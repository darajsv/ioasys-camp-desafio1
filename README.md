## Desafio Ioasys Camp 2021

### Descrição

<p> Funcionalidade de Carrinho de Compras de uma e-commerce </p>

### Funcionalidades:

- Adicionar Produto
- Adicionar uma lista de Produto
- Adiconar Cupom
- retornar a lista de itens do carrinho
- retornar a quantidade total de itens no carrinho
- finalizar a compra
- calcular pagamento parcelado
- resumo da compra

### Funcionalidade extra:

- Remove item do carrinho
